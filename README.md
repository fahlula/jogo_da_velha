# ❌⭕ Jogo da Velha — Web + Python

Divirta-se com uma versão clássica do **Jogo da Velha**, com interface web em **HTML** 🌐 e lógica em **Python** 🐍.  
Um projeto perfeito para praticar **lógica de programação**, **DOM** e **integração front-end/back-end**! 🎉

## 👀 Visão Geral

- 🖥️ Interface amigável no navegador (`index.html`)  
- 🧠 Lógica do jogo implementada em Python (`jogo_da_velha.py`)  
- ⚔️ Dois jogadores se enfrentam alternando entre **X** e **O**  
- 🎨 Possibilidade de personalizar estilos e adicionar animações

## 🧰 Tecnologias Utilizadas

- 🌐 **HTML5** → Estrutura visual do jogo  
- 🐍 **Python** → Regras e lógica do jogo  
- 🎨 **CSS3** (opcional) → Estilização do tabuleiro  
- ⚡ **JavaScript** (opcional) → Interatividade dinâmica
